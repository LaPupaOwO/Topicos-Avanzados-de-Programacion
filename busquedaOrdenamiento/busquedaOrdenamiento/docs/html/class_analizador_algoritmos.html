Descripción detallada

Clase encargada de encapsular la logica de generacion, ordenacion y busqueda de datos.

  • Esta clase permite administrar un contenedor de datos dinamico, realizar pruebas de estres con diferentes dimensiones y medir el impacto computacional de diversos algoritmos.
